# Wireframes (Deskriptif)

## Halaman Daftar Trip
- Header: filter lokasi (otomatis), tombol aksi “Trip Baru”
- Tabel: kolom kode, kendaraan, driver, status, aksi
- Dropdown per baris: Tutup Trip, Detail

## Halaman Tambah Trip
- Kartu kiri: Informasi Trip & Kendaraan (kode, waktu, pilih kendaraan, detail kendaraan, kilometer, status, tujuan, penumpang, driver)
- Kartu kanan: Dokumentasi Kendaraan (unggah foto), Catatan Tambahan
- Footer: tombol Batal, Simpan

## Halaman Tutup Trip
- Kode trip, plat, km awal/akhir, unggah foto kembali
- Footer aksi simpan

Struktur grid responsif dan jarak antar elemen konsisten.
