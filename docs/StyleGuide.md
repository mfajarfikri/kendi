# Kendi UI Style Guide

## Warna
- Primer: Blue 500/600, Aksen: Indigo 600/700
- Status: Emerald (success), Red (error), Yellow (warning)
- Latar: White / Gray-800 (dark)

## Tipografi
- Base: text-base (16px), judul: text-lg–xl sesuai hierarki
- Line-height: 1.5 untuk keterbacaan

## Spasi
- Container: p-6, antar elemen: space-y-6
- Grid gap: 4–6 sesuai konteks

## Komponen
- Input: rounded-lg, border-gray-300, focus:ring-blue-500
- Button: gradient primer, hover subtle, disabled: opacity-50
- Listbox/Combobox: menu rounded, shadow, focus state jelas

## Interaksi
- Transition: ease-in-out 150–200ms
- Focus visible: ring-blue-500

## Aksesibilitas
- Kontras teks terhadap latar, label eksplisit, keyboard navigation

## Responsif
- Grid: sm:grid-cols-2, fallback satu kolom di mobile
