# Prototipe Interaktif

## Alat
- Gunakan Figma untuk wireframe → mockup → prototipe interaktif

## Langkah
1. Import Style Guide (warna, tipografi, komponen)
2. Buat frame untuk halaman utama (Daftar, Tambah, Tutup Trip)
3. Tambahkan interaksi: hover, fokus, transisi
4. Jalankan sesi uji pengguna ringan (5–8 orang)

## Usability Testing
- Tugas: tambah trip, tutup trip, cari driver/kendaraan
- Metode: think-aloud, waktu penyelesaian, error rate

## Target
- Engagement +25%, NPS +15, waktu tugas -30%, WCAG 2.1 AA
